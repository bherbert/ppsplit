# Quick Start

1. Create a session folder (e.g., `runs/2025-09-21/`)
2. Right-click the folder → **1) Fetch YouTube video** — paste the URL when prompted
3. Right-click the folder → **2) Create snippets CSV file** — edit `snippets.csv.txt` with one line per clip: `start,end,title`
4. Right-click the downloaded `.mp4` → **3) Extract snippets from video**
   - A dialog will prompt: **"Enable fade in/out transitions?"** — choose Yes to add 1-second fade-in/out on each clip

Clips land in the session folder. Check `ppsplit.log` for the run summary.
